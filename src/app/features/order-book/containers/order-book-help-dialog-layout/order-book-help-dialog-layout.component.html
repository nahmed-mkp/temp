**Please read the below instructions for opertions on the Order Book Screen:

ADD NEW ORDERS

To add a new order to Order Book, click on ADD ORDER button. A flyout will appear on the right of the screen. Populate all the fields for the order and hit save.

EDIT EXISTING ORDERS

If you need to edit an existing order, select the order on the orderbook and lock the order by checking the lock checkbox. Checking the lock on the order does the follow:

a) Brings the order into edit mode and opens a flyout on the right of the screen with all fields for the order populated. Edit the order and hit Save.

b) After the order is saved, the flyout disappears and order is moved to an unlocked state.

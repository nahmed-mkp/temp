Please read the below instructions for adding, updating and deleting Futures Commissions:

ADD NEW COMMISSIONS

  1. Download the Commissions export by clicking on "Download Commissions"
  2. Add rows to the Excel sheet, and for each of these rows, put the word "Add", "Insert" or "New" in the Action column
  3. Delete other records that do not need to be modified
  4. Click the "Upload Commissions" button and point to the file you have just edited

EDIT EXISTING COMMISSIONS

  1. Download the Commissions export by clicking on "Download Commissions"
  2. Edit rows in the Excel sheet. Do not modify the Id columns. For all the edited rows, put the word "Edit", "Update" or "Modify" in the Action column
  3. Delete other records that do not need to be modified
  4. Click the "Upload Commissions" button and point to the file you have just edited

DELETE COMMISSIONS

  1. Download the Commissions export by clicking on "Download Commissions"
  2. In the Action column, put the word "Delete", "Cancel" or "Erase" in all the rows that need to be deleted
  3. Delete other records that do not need to be modified
  4. Click the "Upload Commissions" button and point to the file you have just edited
All of the above steps can be performed as part of a single upload.
